### 第一題

* `iomanip` 是 C++ 管理 `格式化輸出` 的函式庫

```c++
#include<iostream>
#include<iomanip>
using namespace std;
int main()
{
    double a,b;
    cin>>a>>b;
    double result = a/b;    //1.623 -> 1.63
    cout<<setprecision(2)<<fixed<<result+0.004;
}
```

### 第二題
```c++
#include<iostream>
using namespace std;
int main()
{
    int a,b,c,d;
    cin>>a>>b>>c>>d;
    cout<<(c-a)*(d-b)<<endl;
    cout<<(c-a+d-b)*2;
}
```
